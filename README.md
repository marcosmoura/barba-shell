# Barba Shell

Barba Shell is a macOS-only Tauri Application that provides a minimal Desktop Window Manager experience. It has a tiling window manager, a status bar and keybindings shortcuts manager.

- [VS Code](https://code.visualstudio.com/)
- [Tauri VS Code extension](https://marketplace.visualstudio.com/items?itemName=tauri-apps.tauri-vscode)
